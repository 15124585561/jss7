Name
	linkset delete

SYNOPSIS
	linkset delete <linkset-name>

         <linkset-name>
 		is the name of the Linkset to be deleted.

DESCRIPTION
	This command is used to delete an existing Linkset.

EXAMPLE
	
	linkset delete m3ualinkset1
	- The above command will delete the Linkset identified by the name m3ualinkset1.

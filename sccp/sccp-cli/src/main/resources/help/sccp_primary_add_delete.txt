Name
	sccp primary_add delete

SYNOPSIS
	sccp primary_add delete <id>

DESCRIPTION
	This command is used to remove a previously defined primary address.

PARAMETERS

	Standard Parameters
                                             
	<id>		-	The identifier of the address that is being deleted.

EXAMPLES
	sccp primary_add delete 1 

	The above command will delete the primary address identified by the number '1'.
